# ファイルの分割方法

  - function_exists
    - 指定した関数が定義されている場合に true を返す

例1

```
<?php
if (function_exists('imap_open')) {
    echo "IMAP 関数が利用可能です。<br />\n";
} else {
    echo "IMAP 関数は利用できません。<br />\n";
}
?>
```

## requireとincludeの違い
  - require
    - 絶対必要なファイルを読み込むときに使用する
    - 関数
    - 致命的なエラー
  - include
    - 最悪なくても良い
    - HTMLのようなテンプレートファイル
    - Warning

## パスの書き方
  - マジック定数
    - 絶対パスに使用

## dirname
  - 親ディレクトリのパスを返す

```
echo dirname(__FILE__);
```
